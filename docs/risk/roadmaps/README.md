# Risk Roadmaps

Diese Sammlung enthält strategische Roadmaps für die Entwicklung der Risk-Management-Komponenten:

- `RISK_LAYER_ROADMAP.md` — Master-Roadmap für Risk Layer (VaR/CVaR, Validation, Stress/MC, Defense-in-Depth)
- `COMPONENT_VAR_ROADMAP_PATCHED.md` — Roadmap für komponentenbasiertes VaR
- `PORTFOLIO_VAR_ROADMAP.md` — Roadmap für Portfolio-VaR-Implementierung

## Verwendung

Diese Roadmaps dienen als zentrale Referenz für:
- Architekturentscheidungen im Risikomanagement
- Priorisierung von Features und Verbesserungen
- Tracking von Implementierungsmeilensteinen
- Koordination zwischen verschiedenen Risk-Komponenten
