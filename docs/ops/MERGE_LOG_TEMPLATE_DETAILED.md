## Usage

This template is generated by:

```bash
python3 scripts/ops/new_merge_log.py --pr XXX --template detailed
```

**Key Features:**

- **Structured Changes**: Separate Added/Updated/Removed subsections for clarity
- **Verification Notes**: Explicit section for constraints and environment notes
- **Operator How-To Subsections**: Clear separation of usage steps and smoke checks
- **Safety Prompts**: Explicit reminders about gate-safety, path hygiene, and secrets

**When to Use:**

- Complex PRs with multiple types of changes
- PRs requiring detailed operator guidance
- PRs with nuanced verification or risk considerations
- When you want explicit structure and safety reminders
